# TIC-TAC

Tic Tac Toe Game built with Create React App.

[LIVE VERSION](https://heyitssilvia-tic-tac.netlify.app/).

## Stack

👩🏻‍🎨 Designed in Figma

👷🏻‍♀️ Built with React using React Router DOM

🎨 Styled with [Sass](https://sass-lang.com/)

👩🏻‍💻 Deployed with Netlify.

## Project setup

```
npm install
```
